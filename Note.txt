JSON Server Creation
---------------------
    > Create a folder inside your project for server
    > Open the folder and open cmd
    > give command "npm init -y"
    > give command "npm i" for node modules
    > install json-server "npm i json-server"
    > open the whole file in vs code
    > create index.js file
    > Import json server into index.js => const jsonserver = require('json-server');
